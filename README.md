# Shrewd IoT-Centred Instant Asthma Due to Dust Risk Predictor

## Overview

Shrewd is an IoT-based healthcare monitoring system designed to predict asthma risk caused by dust exposure using real-time environmental and physiological monitoring.

The system combines sensor data, wireless communication, and edge computing to provide instant risk assessment and user alerts.
